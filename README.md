# Hyderabad Restaurant Reviews Analysis

This project analyzes restaurant reviews for the Hyderabad Tourism Board using text mining techniques to gain insights into customer experiences.

## Project Goals
- **Cuisine Classification:** Classify restaurants by cuisine type using review content (multilabel classification).  
- **Score Prediction:** Predict Zomato scores based on review sentiment (sentiment analysis).  
- **Additional Analysis:** Identify frequently co-mentioned dishes and extract main themes such as service, ambiance, and food quality.

## Methodology
The project follows the **CRISP-DM framework**, which includes:
1. Business Understanding  
2. Data Understanding  
3. Data Preparation  
4. Modelling  
5. Evaluation  
6. Deployment  

This structured approach ensures thorough analysis and reliable results.

## Tools & Libraries
- Python 3  
- Pandas, NumPy, Matplotlib  
- Scikit-learn  
- NLTK / SpaCy (for text processing)
